# Asch CLI

A command line interface for bootstrapping and managing [Asch](https://asch.so/) blockchain apps.

## Installation:

```
npm install -g asch-cli
```

## Apps

### Add app

```
asch-cli dapps -a
```

### Deposit

```
asch-cli dapps -d
```

### Withdraw

```
asch-cli dapps -w
```

## Contracts

### Add contract

```
asch-cli contract -a
```

### Delete contract

```
asch-cli contract -d
```

## Crypto

### Generate public key

```
asch-cli crypto -p
```

### Generate accounts

```
asch-cli crypto -g
```

### Help

```
asch-cli -h
```
